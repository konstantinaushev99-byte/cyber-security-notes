# Как решил это Nmap?
1. Создаем обычный текстовый файл target.txt
2. Внутри:
   ```bash
   192.168.3.10
   192.168.3.15
   192.168.3.20
   192.168.3.50
   192.168.3.100
3. И запускаем:
   ```bash
   sudo nmap -iL targets.txt
4. Что означает -iL?
   - Input List
   - Тоесть: возьми список целей из файла

---

# --exclude 
Что это - если буквально, то это просто исключить какой-то ip
Например:
```bash
nmap -iL targets.txt --exclude 192.168.3.12
Читается как: Возьми цели из файла (-iL targets.txt), но исключи этот IP (--exclude 192.168.3.
