# --min-rate <значение>
Что это - --min-rate 500 значит что nmap будет отправлять не менее 500 пакетов в секунду
# --max-rate <значение>
Что это - --max-rate 1000 значит что nmap будет отправлять не больше 1000 пакетов в секнду

---

# Пример:
```bash
Без параметров:
sudo nmap -sS 192.168.3.116
Nmap сам решает:
120 pps
↓
300 pps
↓
700 pps
↓
400 pps
```
И теперь с параметрами:
1. sudo nmap -sS --min-rate 1000 ( В этот момент мы просим Nmap отправлять не меньше 1000 пакетов, если это позволяет сеть )
   ```bash
   1000 pps
   ↓
   1300 pps
   ↓
   1700 pps
   ↓
   2100 pps
2. sudo nmap -sS --max-rate 1000 ( В этот момент мы просим Nmap отправлять не больше 1000 пакетов )
   ```bash
   200 pps
   ↓  
   600 pps
   ↓
   950 pps
   ↓
   1000 pps
