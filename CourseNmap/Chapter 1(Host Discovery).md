# Host Discovery - обноружение хостов
### Какие устройства вообще сейчас находятся в сети? - именно это называется Host Discovery

# Что делает команда - nmap -sn 192.168.3.0/24
### Что означает опция -sn - не сканировать порты, только определить, какие хосты живы

# Но как он это понимает?
### Вариант 1 - устройство в твоей локальной сети
- Nmap не будет использовать ICMP, сначало он отправит ARP Request
  ```bash
  Who has 192.168.3.94?
  Tell 192.168.3.109
- Телефон отвечает:
  ```bash
  192.168.3.94 is at
  E2:4A:C3:E3:41:CC
