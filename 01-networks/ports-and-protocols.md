# Itroduction to IP

IP(Internet Protocol) - отвечает за доставку пакетов от отправителя до получателя 
Есть два основных протокола:
1. TCP - надежная доставка данных без потерь(только с подтверждение), примеры использования: Веб-сайты,почта,файлы
2. UDP - отправка без  потверждения(быстрый, но не надежный),Видиозвонки,DNS-запросы
