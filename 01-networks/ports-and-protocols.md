# Itroduction to IP

IP(Internet Protocol) - отвечает за доставку пакетов от отправителя до получателя 
Есть два основных протокола:
1. TCP - надежная доставка данных без потерь(только с подтверждение), примеры использования: Веб-сайты,почта,файлы
2. UDP - отправка без  потверждения(быстрый, но не надежный),Видиозвонки,DNS-запросы

## Other Useful Protocols

ICMP (Internet Control Message Protocol) — «проверка связи».
- Команда ping использует ICMP 
- Сообщает об ошибках («хост недоступен», «TTL истёк»)

GRE (Generic Routing Encapsulation) — создаёт туннель между двумя точками.
- Используется в VPN
- Сам по себе не шифруетданные 

IPSec (Internet Protocol Security) — шифрует данные в туннеле (VPN).
- Обеспечивает конфиденциальность, целостность и аутентификацию 
- Два режима: Transport Mode (шифрует только данные) и Tunnel Mode (шифрует всё)
- Два протокола: AH (аутентификация) и ESP (шифрование + аутентификация)
- Процесс установки: IKE (Internet Key Exchange) через UDP port 500 

Для SOC важно: IPSec — стандарт для безопасных VPN. GRE часто используется с IPSec.

## Network Communication (Unicast, Multicast, Anycast, Broadcast)

| Тип | Что значит | Пример | IPv6? |
|:----|:-----------|:-------|:------|
| **Unicast** | Один → Одному | Обычный запрос к сайту | ✅ Да |
| **Broadcast** | Один → **Всем** в локальной сети | ARP-запрос: «Кто владелец IP X?» | ❌ Нет (заменён на multicast) |
| **Multicast** | Один → **Подписавшимся** | Видеотрансляция, обновления маршрутов | ✅ Да |
| **Anycast** | Один → **Ближайшему** из многих | DNS (запрос идёт к ближайшему серверу) | ✅ Да |

**Важно для SOC:**
- **Broadcast** ограничен локальной сетью (не проходит через роутер) [citation:4]
- ARP и DHCP работают через Broadcast [citation:7]
- В IPv6 Broadcast **отсутствует** — вместо него multicast [citation:4]

**Как запомнить:**
- **Unicast** — персональное письмо
- **Broadcast** — объявление по громкоговорителю для всех
- **Multicast** — рассылка только для подписчиков
- **Anycast** — звонок в службу поддержки: отвечает ближайший оператор
