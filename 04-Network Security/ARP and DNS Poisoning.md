# ARP poisoning (ARP spoofing)

Как работает ARP?
'bash
Представим:
ПК: 192.168.1.9
Роутер: 192.168.1.1
Компьютер знает IP роутера:
192.168.1.1
Но не знает его MAC-адрес.
Поэтому отправляет ARP-запрос:
Кто имеет IP 192.168.1.1?
Сообщите свой MAC-адрес.
