# Social Engineering

---

# Что такое социальная инженерия
Атака на человека, а не на технику. Злоумышленник манипулирует людьми, чтобы получить доступ, пароли или информацию.

---

# Типы атак

| Тип | Что делает | Пример |
|:----|:-----------|:-------|
| Tailgating | проникновение в здание вслед за авторизованным человеком | Придержите дверь, я забыл пропуск |
| Dumpster Diving | поиск полезной информации в мусоре компании | Найти выброшенные бумаги с паролями | 
| Phishing | Массовая рассылка поддельных писем | Ваш аккаунт заблокирован, перейдите по ссылке |
| Spear phishing | Целевой фишинг (на конкретного человека) | Письмо от директора для бухгалтера |
| Whaling | Фишинг на топ-менеджеров (CEO, CFO) | Срочно переведите 50 000$ на счёт X |
| Pretexting | Атакующий создаёт легенду (предлог) | Здравствуйте, я из техподдержки, скажите пароль |
|  Shoulder surfing | Подглядывание за экраном/клавиатурой | В кафе смотрят, как вы вводите пароль |

--- 

# Как защищаться

| Против | Защита |
|:-------|:-------|
| Phishing | Обучение сотрудников, не переходить по ссылкам из писем |
| Tailgating | Пропускная система, не впускать незнакомцев |
| Shoulder surfing | Приватность экрана, не вводить пароли на людях |
| Pretexting | Всегда перезванивать по официальному номеру |

---

# Для собеседования 

Фишинг — массовые поддельные письма
Spear phishing — целевой
Tailgating — зайти в офис следом
Pretexting — легенда для обмана
Shoulder surfing — подглядывание
