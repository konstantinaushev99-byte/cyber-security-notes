# Device Security(Безопасность устройств)

---

# Лучшие практики (Best Practices)

# Default Credentials (Стандартные пароли)
- Многие устройства поставляются с заводскими логинами и паролями
  - Например: admin/admin
- Проблема: Если пароль не изменить, любой человек, знающий стандартные данные, может получить доступ

# Port Security
- Функция коммутатора (Switch), которая защищает порты от подключения чужих устройств
- Как работает
   - Коммутатор запоминает MAC-адрес устройства
   - Например:
     - Port 1 -> MAC AA:BB:CC:DD:EE:FF
   - Если кто-то отключит компьютер и подключит свой ноутбук:
     - Новый MAC -> Нарушение политики -> Блокировка порта

# Disable Unused Ports
- Отключение неиспользуемых порто
- Например: В переговорной есть Ethernet-розетка.
  - Если она не используется: Switch Port -> Disabled
- Преимущества:
  - злоумышленник не сможет просто подключиться
  - уменьшается риск несанкционированного доступа

# MAC Filtering
- Ограничение доступа по MAC-адресу
- Разрешённые устройства:
  - AA:BB:CC:11:22:33
  - DD:EE:FF:44:55:66
- Если MAC нет в списке:
  - Access Denied
- Недостаток MAC Filtering
 - MAC-адрес можно подделать
 - Это называется MAC Spoofing
 - Поэтому MAC Filtering не считается надежной защитой

---
  # MAC Spoofing
- это изменение MAC-адреса сетевого адаптера, чтобы устройство выдавало себя за другое устройство в сети
- Допустим, в сети разрешён только MAC: AA:BB:CC:DD:EE:FF
- Атакующий может:
  - Узнать этот MAC-адрес
  - Изменить свой MAC на такой же
  - Получить доступ в сеть

---
  
# Nmap
- Инструмент аудита сети
- Используется для обнаружения открытых портов и сервисов
- Помогает проверять поверхность атаки устройства
    
---

# Security Through Obscurity
- Безопасность через сокрытие
- Основана на том, что злоумышленник не знает, как работает защита
- Не считается надежной защитой
- Пример: MAC Filtering

---

# NAC (Network Access Control)
- Контроль доступа к сети
- Проверяет устройство перед выдачей доступа

---

# 802.1X
- Стандарт NAC
- Использует аутентификацию пользователя или устройства
- Часто применяется в корпоративных сетях
