# Поиск
1. grep - Ищет строки, содержащие заданный текст (или шаблон).
   | Команда | Результат |
   |:--------|:----------|
   | grep "error" log.txt | Показывает строки с "error" |
   | grep -i "warning" log.txt | Без учёта регистра (Warning, WARNING) |
   | grep -n "error" log.txt | Показывает номера строк |
   | grep -c "error" log.txt | Показывает количество совпадений |
   | grep -E "Failed|Invalid" auth.log | Ищет Failed или Invalid |
   | grep -E "^192\.168\." access.log | Строки, начинающиеся с 192.168. |
   | grep -E "error$" log.txt | Строки, заканчивающиеся на error |

2. find - ищет файлы и папки по заданным критериям (имя, размер, дата, тип)
   
  2.1 Базовое использование
  | Команда | Результат |
  |:--------|:----------|
  | `find . -name "file.txt"` | Ищет файл с точным именем `file.txt` в текущей папке (`.` — текущая директория) |
  | `find /home -name "*.md"` | Ищет все `.md` файлы в домашней папке |
  | `find ~ -type d -name "notes"` | Ищет папку `notes` в домашней директории |
  | `find . -iname "CONFIG"` | Без учёта регистра (найдёт `config`, `Config`, `CONFIG`) |

  2.2 Ключевые критерии поиска
  По имени (-name)
 | Команда | Что ищет |
 |:--------|:---------|
 | `find . -name "*.log"` | Все файлы с расширением `.log` |
 | `find . -name "file?"` | `?` заменяет ровно **один** символ (`file1`, `fileA`) |
 | `find . -name "file*"` | `*` заменяет **любое** количество символов (`file1.txt`, `file-backup.zip`) |

  2.3 По типу (-type)
 | Опция | Тип | Пример |
 |:------|:----|:-------|
 | `-type f` | Обычный файл | `find . -type f -name "*.sh"` |
 | `-type d` | Папка (directory) | `find . -type d -name "backup"` |
 | `-type l` | Символическая ссылка | `find . -type l` |

 2.4 По размеру (`-size`)
 | Команда | Что ищет |
 |:--------|:---------|
 | `find . -size +100M` | Файлы **больше** 100 МБ |
 | `find . -size -1M` | Файлы **меньше** 1 МБ |
 | `find . -size 100M` | Файлы ровно 100 МБ |
 Единицы размера: `c` (байты), `k` (килобайты), `M` (мегабайты), `G` (гигабайты).

 2.5 По дате изменения (`-mtime` — modification time)
 | Команда | Что ищет |
 |:--------|:---------|
 | `find . -mtime -7` | Файлы, изменённые **за последние 7 дней** |
 | `find . -mtime +30` | Файлы, изменённые **более 30 дней назад** |
 | `find . -mtime 0` | Файлы, изменённые **сегодня** |
 | `find . -mmin -60` | Файлы, изменённые **за последние 60 минут** |

 2.6 Действия над найденным

 | Действие | Что делает | Пример |
 |:---------|:-----------|:-------|
 | `-print` | Вывести путь (по умолчанию) | `find . -name "*.log" -print` |
 | `-ls` | Подробный вывод (как `ls -l`) | `find . -name "*.log" -ls` |
 | `-delete` | **Удалить** найденные файлы | `find . -name "*.tmp" -delete` |
 | `-exec` | Выполнить команду для каждого файла | `find . -name "*.txt" -exec cat {} \;` |
