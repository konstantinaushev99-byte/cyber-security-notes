# locate — мгновенный поиск файлов по имени
 Мгновенно находит файлы по имени, используя предварительно созданную базу данных.

Отличие от find:
| Команда | Как ищет | Скорость | Видит новые файлы? |
|:--------|:---------|:---------|:-------------------|
| find | В реальной файловой системе | Медленно |  Да (сразу) |
| locate | По базе данных (updatedb) | Мгновенно |  Нет (до обновления БД) |

База данных обновляется автоматически раз в сутки (обычно через cron). Свежеустановленная система может не иметь базы данных, её нужно создать вручную.

 Базовое использование
| Команда | Результат |
|:--------|:----------|
| locate passwd | Находит все файлы, в имени которых есть "passwd" |
| locate -i pasSwd | Поиск без учёта регистра |
| locate -b passwd | Ищет только в имени файла (без пути) |
| locate -c passwd | Показывает только **количество** найденных файлов |
| locate -l 5 passwd | Ограничивает вывод 5 результатами |

# which — поиск пути к программе
Описание: Показывает полный путь к исполняемому файлу (программе), который будет запущен при вводе команды.

 Базовое использование
| Команда | Результат |
|:--------|:----------|
| which ls | Показывает, где находится ls (обычно /usr/bin/ls) |
| which python | Показывает путь к интерпретатору Python |
| which grep awk sed | Показывает пути для нескольких команд сразу |
