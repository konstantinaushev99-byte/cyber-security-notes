# cut 
### Что делает - вырезание столбцов/полей

Основные флаги:
- -d — задать разделитель (delimiter)
- -f — какое поле (field) вывести
- -c — вырезать по номеру символа (character), а не по полю

Примеры:
1. Разделитель :, берём 1-е поле
```bash
echo "root:x:0:0:root:/root:/bin/bash" | cut -d':' -f1
Вывод: root
```
2. Можно указать несколько полей через запятую
```bash
echo "root:x:0:0:root:/root:/bin/bash" | cut -d':' -f1,7
Вывод: root:/bin/bash
```
3. Диапазон полей через дефис
```bash
echo "root:x:0:0:root:/root:/bin/bash" | cut -d':' -f1-3
Вывод: root:x:0
```
4. По символам — первые 3 символа
```bash
echo "Hello" | cut -c1-3
Вывод: Hel
```

# awk
### Что делает - работает построчно, автоматически разбивает каждую строку на поля (по умолчанию — по пробелам/табам, независимо от их количества) и даёт доступ к ним через $1, $2, $3... где $0 — вся строка целиком

Базовый синтаксис:
```bash
awk '{print $1}' файл
```

### Пример:
1. Заметьте — множественные пробелы не мешают, в отличие от cut
```bash
echo "  root    x   0   0" | awk '{print $1}'
Вывод: root
```
2. -F задаёт разделитель, как -d у cut
```bash
cat /etc/passwd | awk -F':' '{print $1}'
Выведет первое поле в passwd
```
3. Несколько полей через запятую — awk сам вставит пробел между ними
```bash
awk -F':' '{print $1, $7}' /etc/passwd
Выведет первое и седьмое поле в passwd
```

### Условия (фильтрация строк) — вот где awk сильнее cut:
1. Выведет IP только из строк, где есть слово "open"
```bash
cat NmapScan2.gnmap | awk '/open/ {print $2}'
Выведет все ip где есть слово 'open'
```
2. Выведет строки, где 3-е поле больше 100 (числовое сравнение)
```bash
awk '$3 > 100' файл.txt
```
3. Пример посложнее — вытащить IP и открытые порты из gnmap:
```bash
grep "open" NmapScan2.gnmap | awk -F'\t' '{print $2, $4}'
```

Встроенные переменные:
1. NR — номер текущей строки
2. NF — количество полей в строке
3. $NF — последнее поле в строке

#### Пример:
```bash
awk '{print NR, $0}' файл
нумерует строки

awk '{print $NF}' файл
печатает последнее поле каждой строки
```

# Когда использовать
| Задача | Инструмент |
|--------|------------|
| Простое извлечение по фиксированному разделителю | cut |
| Множественные/непостоянные пробелы | awk |
| Нужны условия/фильтры | awk |
| Нужна математика, счётчики, сложная логика | awk |
| Быстро и просто, без изысков | cut |
