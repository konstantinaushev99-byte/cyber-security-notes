# Sort and uniq
### Что делает sort - сортирует строки

- если это текст - по афавиту
- если это числа - лексикографически (как текст)

Для числовой сортировки используют:
```bash
sort -n scan.txt
```

Например:
Файл (scan.txt)
```bash
22
443
53
22
80
3389

Команда: sort scan.txt

22
22
3389
443
53
80
```

---

# uniq
### Что делает uniq - удаляет соседние одинаковые строки

Например:
```bash
Файл(scan.txt)
apple
apple
banana
banana
orange

Команда: uniq scan.txt

apple
banana
orange
```

Если строки будет стоять не вместе, то нечего не поменяется, поэтому их используют вместе:
```bash
sort scan.txt | uniq
```
