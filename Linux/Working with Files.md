# Работа с файлами

1. touch - создаёт пустые файлы или обновляет временные метки существующих
   | Команда | Результат |
   |:--------|:----------|
   | touch file1.txt | Создаёт пустой файл file1.txt |
   | touch notes.md script.sh | Создаёт сразу два файла |
   | touch ~/Documents/log.txt | Создаёт файл в папке Documents |

2. mkdir - создаёт новую директорию (папку/каталог)
   | Команда | Результат |
   |:--------|:----------|
   | mkdir NewFolder | Создаёт папку NewFolder в текущей директории |
   | mkdir my-notes | Создаёт папку my-notes |
   | mkdir "Folder with spaces" | Если в имени нужны пробелы — берём в кавычки |

3. cp - копирует файлы или папки из одного места в другое
   | Команда | Результат |
   |:--------|:----------|
   | cp file1.txt file2.txt | Копирует file1.txt → file2.txt (создаёт новый файл) |
   | cp file.txt folder/ | Копирует файл в папку folder |
   | cp file.txt ~/Documents/ | Копирует в домашнюю папку Documents |
   | cp -r folder1/ folder2/ | Копирует папку folder1 со всем содержимым в folder2 |

4. mv - перемещает файлы/папки или переименовывает их
   | Команда | Результат |
   |:--------|:----------|
   | mv file.txt folder/ | Перемещает file.txt в папку folder |
   | mv file1.txt file2.txt folder/ | Перемещает несколько файлов в папку |
   | mv file.txt folder/file1.txt | Перемещает и переименовывает file.txt в папку folder с названием file1.txt |
   | mv -i file.txt folder/ | Спрашивает подтверждение, если файл уже существует в папке |

5. rm - удаляет файлы или папки
   | Команда | Результат |
   |:--------|:----------|
   | rm file.txt | Удаляет file.txt |
   | rm file1.txt file2.txt | Удаляет несколько файлов |
   | rm -r folder/ |  удаляет папку со всем содержимым |
   | rm -i file.txt | Спрашивает подтверждение перед удалением |
   | rm -rf folder/ | Удаляет папку без вопросов (опасно!) |

6. head - Показывает первые строки файла
   | Команда | Результат |
   |:--------|:----------|
   | head file.txt | Показывает первые 10 строк |
   | head -n 20 file.txt | Показывает первые 20 строк |
   | head -5 file.txt | Показывает первые 5 строк |

7. tail - показывает последние строки файла
   | Команда | Результат |
   |:--------|:----------|
   | tail file.txt | Показывает последние 10 строк |
   | tail -n 20 file.txt | Показывает последние 20 строк |
   | tail -5 file.txt | Показывает последние 5 строк |
   | tail -f file.txt | Follow — показывает новые строки в реальном времени (Ctrl+C для выхода) |

8. less - показывает файл по страницам. Можно листать вверх/вниз, искать, выходить
   less file.txt -  Открывает файл в постраничном режиме
   Навигация:
   | Действие | Что происходит |
   |:--------|:----------|
   | Пробел или Page Down | Следующая страница |
   | b или Page Up | Предыдущая страница |
   | Вниз / Вверх | По одной строке |
   | /слово | Искать вперёд (нажимайте n для следующего совпадения) |
   | ?слово | Искать назад |
   | q | выйти |

9. cat -  выводит содержимое файла (или нескольких файлов) в терминал
   | Команда | Результат |
   |:--------|:----------|
   | cat file.txt | Показывает содержимое file.txt |
   | cat file1.txt file2.txt | Показывает сначала file1.txt, потом file2.txt |
   | cat -n file.txt | Показывает с номерами строк |

# Итог
1. touch - создаёт пустые файлы или обновляет временные метки существующих
2. mkdir - создаёт новую директорию (папку/каталог)
3. cp - копирует файлы или папки из одного места в другое
4. mv - перемещает файлы/папки или переименовывает их
5. rm - удаляет файлы или папки
6. head - Показывает первые строки файла
7. tail - показывает последние строки файла
8. less - показывает файл по страницам. Можно листать вверх/вниз, искать, выходить
9. cat -  выводит содержимое файла (или нескольких файлов) в терминал
