# Software Tools

---

# Protocol Analyzer (анализатор протоколов)
- захватывает кадры (frames) в проводной или беспроводной сети
- показывает их в удобном для человека виде
- позволяет анализировать сетевой трафик

1. Wireshark
- Wireshark позволяет:
 - расшифровывать сетевые протоколы
 - анализировать весь сетевой диалог
 - искать ошибки и проблемы в трафике

---

# Port Scanner
- Сканирует открытые порты на удалённом хосте

1. Nmap(Network Mapper) — инструмент сканирования сети
- Позволяет получать информацию о:
  - открытых портах
  - операционной системе
  - версиях приложений и сервисов
 
 ---

# CDP (Cisco Discovery Protocol)
- проприетарный протокол Cisco
  - Позволяет узнать:
    - IP устройства
    - имя устройства
    - номер порта
    - версию ПО
    - VLAN
    - конфигурацию интерфейсо
- Пример CDP
  - Устройство сообщает: 10.1.10.251
  - Интерфейс: GigabitEthernet2
 
---

# LLDP (Link Layer Discovery Protocol)
- LLDP — открытый аналог CDP
- Позволяет получать информацию:
  - MAC-адрес
  - имя устройства
  - VLAN
  - номер интерфейса
- Пример LLDP
  - Устройство: Studio-300-28
  - Интерфейс: GigabitEthernet2
  - VLAN: 10

---

# Speed Test
- При подключении к Интернету полезно знать реальную скорость канала
- Принцип работы:
  - Отправляется большой объём данных
  - Измеряется скорость передачи
- Рассчитывается:
  - Download Speed
  - Upload Speed
 
# Для собеседования
- Protocol Analyzer → захватывает и анализирует сетевой трафик
- Wireshark → самый популярный анализатор протоколов
- Nmap → сканер сети для поиска устройств, портов и сервисов
- Active Scan → отправка запросов и анализ ответов
- CDP → Cisco Discovery Protocol (только Cisco)
- LLDP → открытый аналог CDP
- Speed Test → измерение скорости загрузки и выгрузки данных
- Rogue Device → неавторизованное устройство в сети
