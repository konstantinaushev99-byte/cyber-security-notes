# Command Line Tools – CompTIA Network+ N10-009 – 5.5
# Command Line Tools

---

# Ping - Проверяет доступность устройства в сети
- Показывает:
  - отвечает ли хост
  - задержку (latency)
  - потери пакетов
- Пример: ping 8.8.8.8

---

# Traceroute / Tracert - Показывает маршрут пакета до конечного узла
- Показывает:
  - все промежуточные маршрутизаторы (hop)
  - задержку на каждом участке
- Пример:
  - traceroute 8.8.8.8
 
---

# Nslookup - Запрашивает информацию у DNS-сервера
- Показывает:
  - IP адрес сайта
  - DNS записи
- Пример:
  - nslookup google.com
 
---

# Dig - Современная замена nslookup
- Показывает:
  - A записи
  - AAAA записи
  - MX записи
  - TXT записи
  - TTL
-  Пример:
   - dig google.com

---

# Tcpdump - Анализатор сетевого трафика из командной строки
- Позволяет:
  - захватывать пакеты
  - фильтровать трафик
  - сохранять дампы
- Примет:
  - tcpdump
- Сохранить в файл:
  - tcpdump -w capture.pcap

---

# Wireshark - Графический анализатор сетевого трафика
- Позволяет:
  - видеть пакеты
  - разбирать протоколы
  - фильтровать трафик

---

# Netstat - Показывает сетевые соединения устройства
- Показывает:
  - открытые порты
  - активные соединения
  - состояние соединений
- Пример:
  - netstat -a 
- Полезные ключи:
  - netstat -a - Все соединения
  - netstat -n - Только IP адреса

---

# Ifconfig
- Показывает:
  - IP адрес
  - MAC адрес
  - состояние интерфейса
- Пример:
  - ifconfig

---

# Nmap - Сканер сети и устройств
- Позволяет определить:
  - открытые порты
  - службы
  - операционную систему
  - версии сервисов
- Пример:
  - nmap 192.168.1.10
 
---

# Curl - Отправляет HTTP/HTTPS запросы из командной строки
- Пример:
  - curl https://google.com
  - Получить только заголовки: curl -I https://google.com
  - Отправить POST запрос: curl -X POST https://site.com
 
---

# ARP 
- Показывает соответствие:
  - IP ↔ MAC
- Пример:
  - arp -a

---

# IP Command - Современная замена ifconfig в Linux
- Показывает:
  - интерфейсы
  - IP адреса
  - состояние интерфейсов
- Пример:
  - ip addr

  ---
  
# Для собеседования
- Ping → Проверка доступности хоста
- Traceroute → Показывает маршрут до хоста.
- Nslookup / Dig → Проверка DNS.
- Netstat → Активные соединения и открытые порты.
- ARP → Соответствие IP и MAC.
- Ipconfig / Ifconfig → Просмотр сетевой конфигурации.
- Nmap → Сканирование сети и открытых портов.
- Tcpdump / Wireshark → Захват и анализ пакетов.
- Curl → Проверка веб-сервисов и API.
