   - Какой IP вернулся в ответе? - 74.125.205.x

# Итог
1. Кто отправил DNS-запрос? - Мой компьютер (192.168.3.109)
2. Кто отправил DNS-ответ? - DNS-сервер (192.168.3.1)
3. Зачем нужен DNS? - Для преобразования доменного имени в IP-адрес.
4. Что происходит после получения IP? - Компьютер подключается к серверу по этому IP.
