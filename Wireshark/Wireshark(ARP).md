# ARP - Address Resolution Protocol

---

# Как выглядит ARP
Запрос:
```bash
Who has 192.168.3.94?
Tell 192.168.3.109

Перевод:
Кто владеет IP 192.168.3.94?
Сообщите 192.168.3.109
```
